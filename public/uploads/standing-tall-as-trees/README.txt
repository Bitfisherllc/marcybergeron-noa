Standing Tall as Trees — image folder
======================================

Path on disk (this project):
  website/public/uploads/standing-tall-as-trees/

Public URL prefix (always):
  /uploads/standing-tall-as-trees/

What to do
-----------
1. Copy your prepared JPG / PNG / WebP files into this folder (any names you like).

2. In Admin → Series → “Standing Tall as Trees”:
   - Replace featured image, or add/edit artworks and choose an image file.
   - New uploads are saved under this same folder (using the series slug), with a unique filename.

3. When you use Admin to upload, pick files from this folder in the file dialog if you like—the server stores them here with a unique generated filename. To replace an image, use “Replace image” on the artwork in Admin.

Gallery on the site:
  http://localhost:3000/art/standing-tall-as-trees
